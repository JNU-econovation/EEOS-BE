## 실행방법
1. git clone https://github.com/JNU-econovation/EEOS-BE
2. 환경변수를 설정해줍니다.(env.properties)
3. db를 실행합니다.
  ```shell
  cd resources
  cd local-develop-environment
  docker-compose up
  ```
4. spring boot를 실행합니다.
